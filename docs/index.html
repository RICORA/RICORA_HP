RICORA

PROGRAMMING TEAM

東京理科大学 野田キャンパスでプログラミングを主とした活動をしている公認団体です。
入部条件はコンピューター技術への興味・関心があることだけ！今の技術力は不問です。皆さんの入部をお待ちしています！

活動内容

競技

  • 競技プログラミング

    競技プログラミングでは、参加者全員に同一の課題が出題され、より早く与えられた要求を満足するプログラムを正確に記述することを競います。 計算時間の見積もりやコーナーケースの処理などの能力が求められ、楽しくアルゴリズムを学びプログラミングスキルを高めることができます。 主要なコンテストサイトにはAtCoder、AizuOnlineJudge、Topcoder、Codeforcesなどがあります。

  • CTF: Capture The Flag

    CTF(Capture The Flag)とはコンピュータセキュリティ技術の競技です。 パケット分析、プロトコル解析、システム管理、プログラミング、暗号解読などの知識や技能が試されます。 楽しくセキュリティの知識をつけることができます。

  • Kaggle

    Kaggleとは企業や研究者が投稿したデータでその最適モデルを競い合う競技です。統計学や機械学習の知識や技能が試されます。

  • その他

    現在取り組んでいる部員はいませんが、その他にもWebアプリ高速化コンテストのISUCONやゲームAIコンテストのcodingameなどがあります

開発

  • Web

    Webアプリは大きく分けて、ボタンや装飾などの表側を指すフロントエンドと、Webサーバーやデータベースなどの裏側の部分を指すバックエンドという2つの仕組みによって動いています。 フロントエンドではHTML、CSS、JavaScript、(WebAssembly)などが用いられます。 バックエンドではNode.js、Python、Rubyなどがよく用いられます。

  • ゲーム

    ウィンドウ表示やスプライト管理などゲーム開発において汎用性のあるプログラムをひとまとまりにしたものを(ゲーム)ライブラリといいます。 このゲームライブラリにステージやアニメーションなどを効率良く簡単に作成することができるツールを統合したものをゲームエンジンといいます。 ゲーム開発ではOpenSiv3D(C++)、LWJGL(Java)、Arcade(Python)などのゲームライブラリやUnity(C#)、Unreal Engine(C++)などのゲームエンジンを使用するのが一般的です。

  • 電子工作・IoT

    Raspberry PiやArduinoを用いると、電子部品を本体に接続しプログラミングをすることでロボットやIoT開発を手軽に体験することができます。

  • サーバー

    電子工作でお馴染みのRaspberry PiはOSが搭載されたコンピューターであるため、Webサーバー構築に使用することもできます。 このサイト(tus-ricora.com)は部員の構築したサーバー上で運用されています。 AWS、Azure、GCPなどのクラウドインフラサービスを用いると、必要なときに必要な規模のサーバーを簡単に構築することができます。 このようなサービスでは数多くの機能が提供されています。

  • ブロックチェーン

    ブロックチェーンとは分散型管理台帳のことで、従来の1つのサーバーにデータを保存する中央集権型とは違ってたくさんのデータ管理者がそれぞれデータの整合性を確かめながらデータを保存していくものです。 すでに出来上がっているデータ群にデータを加えていく様から、データをブロック状にチェーンで繋げていく、ということでブロックチェーンという名前がついています。 ブロックチェーン技術を利用したアプリケーション開発を行なっていきます。

発表・その他

  • LT: Lightning Talks

    サークルでは定期的にLTが行われており、サークルメンバーで学んだものを共有します。 LTを通じて、様々な分野のプログラミング技術について知ることができます。 また、自分で発表を行うことでより自分の理解を深めることもできます。

  • 技術書店

    技術書典とは技術書専門の同人誌即売会です。2019年度には技術書典7で合同誌を頒布しました。

  • レクリエーション

    RICORAでは部員の親睦を深めることを目的に定期的にレクリエーションを行っています。 2020年度はMinecraftを用いてレクリエーションを行いました。
