競技プログラミング

競技プログラミングでは、参加者全員に同一の課題が出題され、より早く与えられた要求を満足するプログラムを正確に記述することを競います。

計算時間の見積もりやコーナーケースの処理などの能力が求められ、楽しくアルゴリズムを学びプログラミングスキルを高めることができます。 主要なコンテストサイトにはAtCoder、AizuOnlineJudge、Topcoder、Codeforcesなどがあります。

CTF (Capture The Flag)

CTF(Capture The Flag)とはコンピュータセキュリティ技術の競技です。

パケット分析、プロトコル解析、システム管理、プログラミング、暗号解読などの知識や技能が試されます。 楽しくセキュリティの知識をつけることができます。

Kaggle

Kaggleとは企業や研究者が投稿したデータでその最適モデルを競い合う競技です。

統計学や機械学習の知識や技能が試されます。
